究極ブラウザ・ネットワーク情報監査システム

JavaScript APIおよびWebRTC技術を駆使し、ブラウザが保持・通信しているすべての環境情報を一本化して出力します。

1. ネットワーク環境 ＆ 検出されたIPアドレス（WebRTC）

接続タイプ (Cellular/Wifi等)取得中...
理論上の下り最大速度取得中...
応答速度 (RTT / 往復遅延時間)取得中...
データ保存モード (Save Data)取得中...
WebRTC検出IPアドレス一覧
IPアドレスの検出スキャンを実行中...

2. ハードウェア ＆ 計算資源

論理プロセッサ数 (CPUコア数)取得中...
デバイスメモリ容量 (概算)取得中...
最大タッチポイント数 (マルチタッチ対応)取得中...
デバイスの向き / 傾きセンサー未検出（デスクトップ環境、または静止状態）

3. グラフィックス (GPU) ＆ ディスプレイ詳細

画面の物理解像度 / 有効領域取得中...
デバイスピクセル比 (DPI倍率)取得中...
色深度 / ピクセル深度取得中...
モニターリフレッシュレート (実測)測定中...
GPU ベンダー (WebGL)取得中...
GPU レンダラー (ビデオカード名)取得中...
WebGL 拡張機能サポート数取得中...
HDR (High Dynamic Range) 対応取得中...

4. オーディオ ＆ メディア入出力デバイス (要権限)

利用可能な音声・映像入力機器型番上の「権限要求」ボタンを押して許可してください
オーディオ・サンプリングレート取得中...
出力チャンネル数 (スピーカー構成)取得中...

5. ブラウザストレージの空き容量 ＆ 永続化状態

ブラウザに割り当てられた総容量取得中...
現在の使用量取得中...
永続ストレージ権限 (Persistent Storage)取得中...

6. 環境固有情報（フォント・プラグイン・タイムゾーン）

ローカルタイムゾーン / UTCオフセット取得中...
検出されたシステムフォント検証中...
ブラウザ登録プラグイン一覧
取得中...
